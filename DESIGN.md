## 项目概述

单文件 Canvas 2D 卧室装饰应用，用于治愈和放松。右侧面板自由改造房间风格，点击互动撒花瓣飘爱心，黑板记录心情。

## 文件结构

```
董婷的浪漫小屋/
├── index.html    # 唯一文件，包含 HTML/CSS/JS (~960行)
└── DESIGN.md     # 本文件
```

## 核心功能

| 功能 | 说明 | 触发方式 |
|------|------|----------|
| 🛏️ 3D立体横放床 | 带透视感的床，床头板+床垫+被子+枕头 | 自动渲染 |
| 🖼️ 36色墙壁 | 暖白/粉色/奶油/薰衣草/清新 5大类 | 面板→墙壁→点色块 |
| 🪟 窗户开关 | 窗扇外开动画 | 点击窗户 / 面板切换 |
| 🌤️ 6种天气 | 星空/晴天/下雨/下雪/彩虹/晚霞 | 面板→天气 |
| 🪟 5种窗帘 | 优雅/蕾丝/丝绒/罗马/束带 + 10色 | 面板→窗帘 |
| 📝 心情黑板 | 点黑板选心情，治愈回复 | 点击黑板 / 面板「心情」按钮 |
| 👗 梦幻衣橱 | 5大类42件时髦衣物，可搭配试穿 | 点击衣柜 / 面板 |
| 🐱 治愈猫咪 | 床边睡觉猫 + 点击聊天 + 心情问答 | 点击小猫 / 自动渲染 |
| 🌸 花瓣飘落 | 粉色花瓣旋转飘落 | 自动 |
| ❤️ 爱心特效 | 鼠标移动飘爱心 | 鼠标移动 |
| 💥 点击撒花 | 点击房间撒花瓣+爱心爆炸 | 点击房间任意处 |
| 🎨 改造面板 | 11个分类标签 | 右侧面板 |
| 🎲 随机/重置 | 一键随机混搭或恢复默认 | 面板底部按钮 |

## 颜色体系

### 墙壁 36色 (5大类)
- **暖白系** (8)：`#fdf5f0` ~ `#fefaf7`
- **粉色系** (8)：`#fce4ec` ~ `#ffebee`
- **奶油系** (8)：`#fef8e8` ~ `#fef9ef`
- **薰衣草系** (8)：`#f5f0ff` ~ `#ebe0f0`
- **清新系** (8)：`#f0f8ff` ~ `#f5faf0`

### 主色调
- 背景深色：`#0b0b18`, `#111128`
- 强调粉：`#ffb3ba`, `#d81b60`, `#e91e63`
- 金色点缀：`#d4a574`, `#c9a84c`
- 治愈紫：`#7c4dff`, `#b39ddb`

## 关键状态对象 R

```js
R = {
  wallC, wallPtn,      // 墙壁颜色 + 花纹(floral/none/bold)
  floorT,              // 地板材质(7种)
  ceilS,               // 天花板(4种)
  curtS, curtC,        // 窗帘样式(5种) + 颜色(10色)
  lightS,              // 吊灯(4种)
  weather, winOpen,    // 天气(6种) + 窗户开关
  bedS, bedQC, bedSC,  // 床款/被子色/床单色
  wardS, wardC,        // 衣柜样式/颜色
  rugS,                // 地毯(5种)
  wallDec,             // 墙饰(4种)
  flowerC, flowerN,    // 花色/花量
  mood, moodMsg,       // 心情emoji + 标签
}
```

## 关键函数

| 函数 | 用途 |
|------|------|
| `drawBed()` | 3D横放床：床头板(梯形3D)+床垫厚度+被子+枕头+床腿+月光 |
| `drawWindow()` | 窗户+6种天气渲染+开关动画+窗帘 |
| `drawBlackboard()` | 木框黑板+粉笔字+emoji+粉笔槽板擦 |
| `drawHealingElements()` | 床边猫+Zzz呼吸动画 |
| `updWeatherParticles()` / `drawWeatherParticles()` | 雨滴/雪花粒子 |
| `openMood()` / `closeMood()` | 心情弹窗+8种情绪+治愈回复 |
| `buildPanel()` | 动态构建面板，支持 color_groups 类型 |

## v2.1 更新记录

8. **新增面板分类**：「天气」tab 合并天气+窗户控制
